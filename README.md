# Weather App - React Project

This is a modern **weather application** built with **React** and **Material-UI**.  
It shows the current weather for Casablanca using the **OpenWeather API**, and supports **Arabic and English** translations.

---

## Features

- Display **current temperature, min/max temperatures, and weather description**
- Shows **current date and time** in selected language
- **Language switch** between Arabic & English
- Modern **UI design** with **glassmorphism effect**
- Responsive layout for **mobile and desktop**
